格式化输出

  • Java SE5 推出了 C 语言中 printf() 风格的格式化输出这一功能。
printf()

  • C 语言中的 printf() 并不能像 Java 那样连接字符串，它使用一个简单的格式化字符串，加上要插入其中的值，然后将其格式化输出。

  • printf() 并不使用重载的 “+” 操作符（C 没有重载）来连接引号内的字符串或字符串变量，而是使用特殊的占位符来表示数据将来的位置。而且它还将插入格式化字符串的参数，以逗号分隔，排成一行。

3. String 面试题

    • ```java
      String s = “abc”; // abc 是在字符串常量池中的
      String s1 = “abc”; // 字符串常量池有这样一个特点，当第二次使用的时候，它不会再去新创建，而是拿已有的去用。
      System.out.println(s == s1); // true
      System.out.println(s.equals(s1)); // true
    • ```java
      String s = “abc”;
      String s3 = “ab” + “c”; // 常量优化机制，翻译为了 class 文件之后 就变成了 String s3 = “abc”;
      /*
      byte b = 10 + 10;
      不会 因为 10 和 10 都是常量，常量优化机制，翻译为 class 文件之后 就变成了 byte b = 20;
      byte c = 10;
      byte d = c + 10; // 编译报错 因为 c 是变量。
      */
      System.out.println(s == s3); // true
      System.out.println(s.equals(s3)); // true
